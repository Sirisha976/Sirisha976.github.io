Explored multiple design concepts in Blender and finalized an ergonomic model prioritizing user comfort. Smart Vehicle Pollution Monitoring and Control System

Developed a real-time pollution monitoring and control system using an Arduino and MQ-135 gas sensor to detect harmful vehicular emissions such as CO, CO₂, and NO₂. The system provides visual alerts through LEDs, audible warnings via a buzzer, and automatically activates a fan-based filtration unit when pollution exceeds safe thresholds. Real-time gas levels are displayed on an LCD, ensuring continuous air quality monitoring. Designed as a compact, low-cost prototype, the system offers a practical solution for reducing vehicle-related air pollution, with future scalability for remote data logging and GPS integration.

BAJA SAE All-Terrain Vehicle Development

Collaborated in the design, analysis, and fabrication of an All-Terrain Vehicle (ATV) for the BAJA SAE India competition. Designed steering and suspension components using SolidWorks and Fusion 360, followed by structural analysis to ensure factor of safety and performance optimization. Assisted in preparing DFMEA and DVP for reliability assessment and coordinated the procurement of outsourced components, ensuring dimensional accuracy through GD&T practices. Contributed to the assembly of the vehicle using welding and joining techniques, culminating in a fully functional ATV prototype built for off-road performance and endurance testing.
