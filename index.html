Projects

Children’s Study Table Design

Created a 3D model of a Children’s study table using Blender software. Developed different types of models and finalized a design focusing on safety, easy handling, mobility, and ample space for books and accessories. The table was fabricated using cutting, welding, and joining techniques. Features include an adjustable height mechanism allowing movement up and down to a certain degree, wheels for mobility, and dedicated storage for bottles, boxes, and study materials, balancing aesthetics with functional utility.

Smart Vehicle Pollution Monitoring and Control System

Developed a real-time pollution monitoring and control system using an Arduino microcontroller and MQ-135 gas sensor. The system detects harmful emissions like CO, CO₂, and NO₂, and triggers actions when thresholds are exceeded. Visual (LEDs), audio (buzzer), and mechanical (fan filtration) responses were integrated. An LCD module displays live pollution levels. The prototype is compact and cost-effective, ideal for vehicle pollution control. Future enhancements include GSM/Wi-Fi modules for remote monitoring and GPS for location-based data tracking.

BAJA SAE All-Terrain Vehicle Development

Contributed to the design, analysis, and fabrication of an All-Terrain Vehicle (ATV) for the BAJA SAE India competition. Components were designed using SolidWorks and Fusion 360, followed by structural analysis ensuring factor of safety and optimization. DFMEA, DVP, and GD&T practices were employed for design validation. The vehicle was assembled using welding and mechanical joining methods with outsourced components. Successfully developed a functional ATV prototype designed for off-road endurance and performance testing.
